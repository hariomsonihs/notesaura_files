Introduction to C Programming

What is C?

C is a high-level, general-purpose programming language that provides low-level access to memory and system processes. It is widely used for system programming, embedded systems, and application development.

  • Low-Level Languages: Closer to machine code (e.g., Assembly).
  • High-Level Languages: Closer to human language (e.g., Python, Java, C).

History of C

Developed by Dennis Ritchie at Bell Labs (1969â€“1973), C evolved from the B language and was used to implement UNIX. The 1978 book by Kernighan and Ritchie standardized C, and ANSI C (C89) and C99 further improved it.

Key Features of C

  • Simple & Efficient: Straightforward syntax, fast execution.
  • Structured Programming: Modular code via functions.
  • Portability: Runs on many platforms with minimal changes.
  • Low-Level Access: Direct memory and hardware manipulation.
  • Rich Library: Standard library for I/O, math, and more.
  • Recursion: Functions can call themselves.

Where is C Used?

  • Operating systems (UNIX, Linux, Windows parts)
  • Embedded systems, device drivers
  • Database engines (MySQL, Oracle core)
  • Compilers, interpreters, and networking
  • Game engines and high-performance apps
Why Learn C?
  • Foundation for C++, Java, Python, and more
  • High performance and control
  • Still in demand for system-level and embedded jobs
  • Large community and resources

Basic Syntax & Structure

  • Preprocessor Directives: e.g., #include
  • Main Function: Entry point (int main())
  • Declarations: Variables and functions
  • Statements: Instructions to execute

Example: Simple C Program

#include <stdio.h> // Preprocessor directive

int main() {
    printf(\"Hello, World!\\n\"); // Print message
    return 0;
}
  • #include <stdio.h>: Includes standard I/O library.
  • int main(): Program entry point.
  • printf: Prints text to the console.
  • return 0;: Ends the program successfully.
Conclusion:

C is a foundational language for modern software. Its speed, portability, and power make it essential for system-level programming and a great starting point for learning other languages.
